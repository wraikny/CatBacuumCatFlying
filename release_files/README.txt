# Cat bacuum cat flying
猫バキューム猫飛んでいる  
Amusement Creators 雙峰祭2019作品  
Author: wraikny  

## Caution
本ゲームによっていかなる不利益がでた場合も、
作者及びAmusementCreatorsは一切の責任の負いません。
ご了承ください。  

画像データをダウンロードするため、インターネット環境が必要です。
その際にセキュリティソフトによってブロックされる場合があります。
自己責任で対応してください。

## Run
### Windows
    CatBacuumCatFlying.exe をダブルクリック

### Mac
    CatBacuumCatFlying.command をダブルクリック

注意: 以下からMono Frameworkのインストールが必要です。  
https://www.mono-project.com/download/stable/  
(あるいは、 `$ brew install mono` を実行する)  

## Requirements
本ゲームを遊ぶためにはインターネット環境が必要です。  
画像データを取得するために[The Cat API](https://thecatapi.com/)を用いています。

* Windows
  * OS : Windows 7以降のOS
  * GPU : GeForce 8400、Radeon HD 2400、Intel HD Graphics 以降のGPU
  * .NET Framework4.7.2

* Mac
  * OS : Mac OS X Mavericks 以降のOS
  * Mono Framework
